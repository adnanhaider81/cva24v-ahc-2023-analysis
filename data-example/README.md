# Synthetic Example Data

The files in this directory are synthetic and are included only to exercise the repository checks. They are not outbreak reads and should not be used for biological interpretation.

- `fastq/CVA24V_DEMO_01_R1.fastq`
- `fastq/CVA24V_DEMO_01_R2.fastq`
- `contigs/CVA24V_DEMO_01_contigs.fasta`

Use private sequencing data by editing `config/config.yaml` and keeping those files outside version control.
